# CWF Admin Reply Brain Change Log

## v0.9 - Batch 009
- เพิ่ม persona แอดมินหญิงสำหรับข้อความตอบลูกค้า ใช้ ค่ะ/นะคะ ให้เป็นธรรมชาติ
- เพิ่มแพทเทิร์นน้ำแอร์ไหลเวลาเปิด โดยเฉพาะเคสคอนโด/ห้องเช่า
- เพิ่มวิธีแนะนำล้างแบบแขวนคอยล์ 1,290 จากเหตุผลเรื่องถาดหลัง/ทางน้ำทิ้ง/ลดปัญหาซ้ำ ไม่ใช่ upsell แข็ง
- เพิ่ม flow วันนี้ไม่ทัน → เสนอพรุ่งนี้มีทีมช่างว่าง → ขอเช็กเวลาสักครู่
- เพิ่ม closing script แบบธรรมชาติสำหรับปิดการจองโดยขอชื่อ เบอร์ ห้อง/ชั้น อาคาร และโลเคชัน
- เพิ่มกติกาให้ข้อความแอดมินสั้นเหมือนคุย LINE จริง ไม่ยาวเป็นบทความ
- เพิ่มข้อห้ามใช้คำลงท้ายผู้ชาย เช่น ครับ/ครับผม ในเทมเพลตแอดมินหญิง
- เพิ่มหลักการว่าไม่ควรฟันธงสาเหตุจากรูป แต่ให้พูดว่า “เบื้องต้นอาจเกิดจาก...”

## v0.8 - Batch 008
- เพิ่มแพทเทิร์นเช็กคิวจากโลเคชันก่อนยืนยันคิววันเดียวกันหรือช่วงเย็น
- เพิ่มหลักการว่าถ้าลูกค้าล้างตามรอบและเลือกแบบปกติ ให้เคารพตัวเลือกนั้น ไม่ดัน upsell เกินจำเป็น
- เพิ่มแพทเทิร์นตอบคำถามเรื่องชำระเงินหลังงานเสร็จ
- เพิ่ม flow เสนอช่วงเวลาที่คอนเฟิร์มได้ก่อน และแจ้งตรง ๆ ถ้าช่วงเวลาที่ลูกค้าต้องการยังต้องรอคอนเฟิร์ม
- เพิ่มการเก็บรายละเอียดคอนโด เช่น เลขห้อง ชั้น อาคาร รวมถึงเบอร์สำรองและโน้ตให้ช่างโทรก่อนมา
- เพิ่มหลักการว่าถ้าลูกค้าไม่ทราบ BTU สามารถให้ช่างดูหน้างานได้
- เพิ่มกติกาแยกรายการหลายเครื่อง/หลาย BTU/หลายประเภทล้าง และส่งใบงานแก้ไขอย่างโปร่งใสเมื่อข้อมูลเปลี่ยน

## v0.7 - Batch 007
- เพิ่มแพทเทิร์นคัดกรองงานซ่อมจากอาการคอยล์ร้อนไม่ทำงาน ใบพัดไม่หมุน มีแต่เสียง หรือขยับแล้วหยุด

- เพิ่มคำอธิบายคำว่า “เนมเพลท” แบบภาษาง่ายสำหรับลูกค้า
- เพิ่มแพทเทิร์นน้ำหยด + กลิ่นอับ และการคัดกรองว่าล้างปกติอาจไม่พอ
