CWF Work Calendar Multi-Select Fix
อัปเดต: 2026-05-07

ไฟล์ใน ZIP นี้เป็นเฉพาะไฟล์ที่แก้ไข:
- tech.html
- app.js
- index.js
- sw.js
- README_WORK_CALENDAR_MULTI_SELECT_FIX.txt

สิ่งที่แก้ไข:
1) ปรับปฏิทินรับงานให้เลือกได้หลายวันก่อนบันทึก
   - แตะวันที่เพื่อเลือก/ยกเลิกเลือก
   - เลือกหลายวันพร้อมกันได้
   - กด “บันทึกวันที่เลือกทั้งหมด” ครั้งเดียว

2) เพิ่มปุ่มเลือกเร็วในปฏิทิน
   - เลือกทั้งเดือน
   - เลือกจันทร์-ศุกร์
   - เลือกเสาร์-อาทิตย์
   - ตั้งค่าเหมือนเดือนก่อน
   - ล้างที่เลือก

3) ตั้งค่ารวมสำหรับวันที่เลือก
   - รับงานล่วงหน้าได้
   - ไม่รับงานล่วงหน้า
   - วันหยุด
   - วันลา
   - เวลาเริ่ม / เวลาเลิก
   - จำนวนงานสูงสุดต่อวัน
   - จำนวนเครื่องสูงสุดต่อวัน
   - หมายเหตุถึงแอดมิน

4) กติกาค่าเริ่มต้นถ้าเลือก “รับงานล่วงหน้าได้” แล้วไม่กรอกเพิ่ม
   - เวลาเริ่ม: 09:00
   - เวลาเลิก: 18:00
   - จำนวนงานสูงสุดต่อวัน: 1 งาน
   - จำนวนเครื่องสูงสุดต่อวัน: 5 เครื่อง
   - หมายเหตุ: เว้นว่างได้

5) ย้ำกติกา Flow
   - ปฏิทินนี้ใช้เฉพาะงานล่วงหน้าที่แอดมินมอบหมาย/บังคับมอบหมายให้
   - ไม่เกี่ยวกับงานด่วน
   - งานด่วนยังใช้ปุ่มเปิด/ปิดรับงานหน้าแรกเหมือนเดิม

วิธีติดตั้ง:
1) แตก ZIP
2) เอาไฟล์ทั้งหมดไปทับใน root โปรเจกต์ CWF
3) commit / push ไป GitHub main
4) รอ Render deploy
5) หลัง deploy ให้ปิด PWA แล้วเปิดใหม่ หรือ clear cache ถ้ายังเห็น UI เก่า

รายการทดสอบหลัง deploy:
1) เข้าแอพช่าง
2) เปิดเมนูแฮมเบอร์เกอร์
3) กด “ตั้งค่าการรับงาน”
4) กด “ตั้งค่าปฏิทินรับงาน”
5) ปฏิทินต้องเปิดได้
6) เลือกหลายวันพร้อมกัน
7) ตั้งค่าเป็น “รับงานล่วงหน้าได้” โดยไม่กรอกช่องเพิ่ม แล้วกดบันทึก
8) ปิดแล้วเปิดใหม่ ต้องเห็นวันที่บันทึกแล้ว
9) ลองเลือกหลายวันแล้วตั้งเป็น “ไม่รับงานล่วงหน้า” ต้องบันทึกได้
10) ลองปุ่ม “ตั้งค่าเหมือนเดือนก่อน” ต้องมีหน้าถามยืนยันก่อน
11) ปุ่มเปิด/ปิดรับงานด่วนหน้าแรกต้องยังใช้งานตาม Flow เดิม

ตรวจสอบ syntax แล้ว:
- node --check index.js ผ่าน
- node --check app.js ผ่าน
- node --check sw.js ผ่าน
