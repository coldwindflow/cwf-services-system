วิธีใช้ (สั้น ๆ)
1) Admin IDs: สร้าง/แก้ไขแอดมิน, เปลี่ยนคอมมิชชั่น(ตัวเลข), เปลี่ยนรหัสผ่าน
2) สวมสิทธิ: เลือกผู้ใช้แล้วกดสวมสิทธิ (มีบันทึก Audit) / หยุดสวมสิทธิ
3) Duration: กำหนดเวลาต่อ service_key (นาที)
4) รายได้ช่าง: ตั้ง Defaults/Override แล้วใช้ “คำนวณตาม Job” เพื่อตรวจผล

💰 รายได้ช่าง (ระบบเดิม: Defaults / Override / คำนวณตาม Job)
หมายเหตุ: ถ้าจะใช้ % ขั้นบันไดตามจำนวนเครื่อง ให้ไปตั้งที่เมนู “ตั้งค่า % ขั้นบันได” และใช้ “งวดจ่ายช่าง” เป็นตัวเลขจริง
เมนูนี้เอาไว้สำหรับระบบเดิม/กรณีเฉพาะเท่านั้น (กันงง)

Override รายบุคคล
เลือกช่าง → เลือกประเภท → ใส่ตัวเลข → บันทึก
- company/custom: ใส่เป็น % ที่ช่างได้จาก “เรทรายการ”
- partner: ใส่ % ที่หักเข้าบริษัท (ช่างได้หลังหัก)
- special_only: ไม่ต้องใส่ตัวเลข (จะถูกมองข้าม)
