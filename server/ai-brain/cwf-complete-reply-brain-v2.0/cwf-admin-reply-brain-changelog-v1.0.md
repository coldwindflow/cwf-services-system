# Change Log — CWF Admin Reply Brain v1.0

## Batch 010
เพิ่มข้อมูลจากผู้ใช้:
- ความแตกต่างของล้างปกติ / พรีเมียม / แขวนคอยล์ / ตัดล้างใหญ่
- เพิ่มกติกาแนะนำแขวนคอยล์หรือตัดล้างใหญ่ทุก 3 ปี
- เพิ่มกติกางานตรวจเช็คอาการแอร์ 700 บาท
- แยกตรวจเช็คเบื้องต้นในงานล้าง ออกจากตรวจเช็คอาการเสียแบบลึก
- เพิ่ม Q&A สำหรับคำถามที่น่าจะเจอบ่อย
- ปรับคำเสี่ยง “จบทุกปัญหา” เป็น “ช่วยลดโอกาสเป็นซ้ำ / ตรวจหาสาเหตุและเสนอแนวทางแก้ไข”
- ล็อก persona แอดมินหญิง ใช้ “ค่ะ / นะคะ”
