# CWF Calendar Button Safe Opener Fix

ไฟล์แก้ไขรอบนี้แก้ปัญหา: กดปุ่ม “ตั้งค่าปฏิทินรับงาน” แล้วไม่มีอะไรเกิดขึ้น

สิ่งที่แก้:
1. เปลี่ยนปุ่มใน tech.html ให้เรียกตัวเปิดแบบ hard-safe โดยตรง
2. เพิ่ม fallback modal ใน tech.html เอง ถ้า app.js/PWA cache/runtime ทำให้ฟังก์ชันหลักไม่ทำงาน ปฏิทินยังเปิดได้
3. fallback modal ใช้งานจริงได้:
   - แตะวันเพื่อเปิด/ปิดรับงานล่วงหน้า
   - รับ/ไม่รับเท่านั้น
   - วันที่มีงานแล้วจะแสดง 📌 มีงาน และล็อกไม่ให้แก้
   - ตั้งเวลา จำนวนงาน จำนวนเครื่อง และหมายเหตุถึงแอดมินได้
4. ปรับ service worker cache version เพื่อบังคับโหลดไฟล์ใหม่

วิธีติดตั้ง:
- แตก ZIP แล้วนำไฟล์ใน ZIP ไปทับ root โปรเจกต์
- commit/push ไป GitHub
- รอ Render deploy สำเร็จ
- ปิด PWA แล้วเปิดใหม่ 1 รอบ หรือรีเฟรชหน้า tech.html

ไฟล์ที่แก้:
- tech.html
- sw.js

ตรวจสอบแล้ว:
- node --check app.js ผ่าน
- node --check index.js ผ่าน
- node --check sw.js ผ่าน
- ตรวจ syntax inline script ใน tech.html ผ่าน
