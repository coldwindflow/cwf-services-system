# CWF AC Error Code Knowledge Base v1.2
อัปเดต: 2026-06-09

ไฟล์นี้เป็นฐานความรู้เริ่มต้นสำหรับ CWF AI Office / Sales AI / Technician AI Assistant  
ใช้เพื่อช่วยแอดมินตอบลูกค้าเมื่อมีอาการเสียหรือแจ้ง Error Code แอร์มา

## กติกาหลัก
- ใช้เป็นข้อมูลคัดกรองเบื้องต้นเท่านั้น ไม่ใช่คำวินิจฉัยสุดท้าย
- ห้าม AI เดาโค้ดที่ไม่มีในฐานข้อมูล
- ถ้าไม่รู้ยี่ห้อ/รุ่น/โค้ด ให้ขอรูปโค้ด รูปรีโมท รูปสติ๊กเกอร์รุ่น และวิดีโอไฟกระพริบก่อน
- ห้ามให้ลูกค้าถอดเครื่อง เปิดฝาบอร์ด จับสายไฟ เติมน้ำยา หรือปีนจุดเสี่ยง
- ถ้าเป็นบอร์ด คอมเพรสเซอร์ น้ำยา ไฟฟ้า สื่อสารเครื่องใน/เครื่องนอก หรือเซนเซอร์ ให้แนะนำตรวจเช็คหน้างาน

## Coverage รอบนี้
- Daikin: วิธีเช็คโค้ดจากแหล่งทางการ แต่ยังไม่แตกตารางความหมายรายโค้ด
- Carrier: วิธีเช็คโค้ด + PDF แหล่งตาราง Ceiling Series แต่ยังไม่ normalize ทุกแถว
- Mitsubishi Electric: ตารางโค้ดเริ่มต้นจากแหล่งทางการ
- Panasonic: ตารางโค้ด common จากแหล่งทางการ
- Samsung: common display/service codes จากแหล่งทางการ
- Haier: ตารางโค้ดจากแหล่งทางการ
- LG: เพิ่ม CH61 จากแหล่งทางการ ยังไม่ใช่ full table

## ข้อความมาตรฐานให้ลูกค้าส่งโค้ด
“รบกวนลูกค้าถ่ายรูปหรือวิดีโอไฟกระพริบ/โค้ดที่ขึ้นให้แอดมินหน่อยนะคะ พร้อมส่งรูปยี่ห้อและรุ่นแอร์ หรือสติ๊กเกอร์ข้างเครื่องถ้าถ่ายได้สะดวกค่ะ โค้ดแต่ละรุ่นอาจแปลไม่เหมือนกัน แอดมินจะช่วยเช็คเบื้องต้นให้ก่อนค่ะ”

## ข้อความเมื่อโค้ดเกี่ยวกับระบบเสียลึก
“โค้ดนี้เบื้องต้นอาจเกี่ยวกับระบบภายในเครื่องนะคะ เช่น เซนเซอร์ บอร์ด มอเตอร์ ระบบน้ำยา หรือการสื่อสารระหว่างเครื่องใน/เครื่องนอก แอดมินแนะนำให้ช่างเข้าตรวจเช็คหน้างานค่ะ ค่าตรวจเช็ค 700 บาท หากมีซ่อมต่อสามารถนำไปรวมกับค่าซ่อมได้ค่ะ”

## ข้อความเมื่อยังไม่มีข้อมูลพอ
“ขอแอดมินเช็คข้อมูลเพิ่มก่อนนะคะ รบกวนลูกค้าส่งยี่ห้อ รุ่นแอร์ รูปโค้ด หรือวิดีโอไฟกระพริบประมาณ 10–20 วินาทีให้หน่อยค่ะ เพราะแต่ละรุ่นโค้ดอาจแปลไม่เหมือนกันค่ะ”
