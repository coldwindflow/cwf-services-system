CWF Work Settings Menu V2

สิ่งที่แก้ในรอบนี้:
1) ย้ายปฏิทินรับงานใหม่ออกจากหน้ารับงานหลักแล้ว เพื่อให้หน้ารับงานไม่รก
2) คงปุ่ม “ตั้งค่าด่วนพื้นที่รับงาน” ไว้ในหน้ารับงานเหมือนเดิม ไม่ย้ายออก
3) เพิ่มปุ่มแรกในเมนูแฮมเบอร์เกอร์ชื่อ “ตั้งค่าการรับงาน”
4) ใน “ตั้งค่าการรับงาน” มี 3 รายการ:
   - ตั้งค่าปฏิทินรับงาน
   - ประเภทงานที่รับได้
   - พื้นที่ประจำของฉัน
5) ในเมนู “ตั้งค่าบัญชี” เอาปุ่มปฏิทิน/ประเภทงาน/พื้นที่ประจำที่กระจัดกระจายออก แล้วเหลือเป็นปุ่มรวม “ตั้งค่าการรับงาน”
6) อัปเดต sw.js/cache version เพื่อบังคับ PWA โหลดหน้าใหม่

ไฟล์ใน ZIP:
- index.js
- app.js
- tech.html
- sw.js

วิธีติดตั้ง:
1) แตก ZIP
2) คัดลอกไฟล์ทั้งหมดไปทับ root repo
3) commit/push ขึ้น GitHub
4) รอ Render deploy
5) หลัง deploy ให้ปิด PWA แล้วเปิดใหม่ หรือ refresh/l้าง cache 1 รอบ

เช็กลิสต์หลัง deploy:
- เปิดแอพช่าง > เมนูแฮมเบอร์เกอร์ ต้องเห็นปุ่มแรก “ตั้งค่าการรับงาน”
- กด “ตั้งค่าการรับงาน” ต้องเห็น 3 รายการ
- กด “ตั้งค่าปฏิทินรับงาน” ต้องเปิดปฏิทินใหม่
- กด “ประเภทงานที่รับได้” ต้องเปิดหน้าตั้งค่าประเภทงาน
- กด “พื้นที่ประจำของฉัน” ต้องเปิดหน้าตั้งค่าพื้นที่ประจำ
- หน้ารับงานหลักต้องยังมีปุ่ม “ตั้งค่าด่วน” สำหรับพื้นที่รับงานด่วน
- หน้ารับงานหลักต้องไม่แสดงการ์ดปฏิทินรับงานใหม่รก ๆ แล้ว
