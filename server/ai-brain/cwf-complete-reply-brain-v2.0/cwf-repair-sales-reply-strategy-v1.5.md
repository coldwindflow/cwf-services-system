# CWF Repair Sales Reply Strategy v1.5
อัปเดต: 2026-06-09

## 1. สรุปการปรับแนวทาง
จากนี้งานซ่อมและ error code ไม่ใช่ “ห้ามตอบ” แต่ให้ AI ตอบแบบ **ตอบเบื้องต้น + เชียร์จองคิวตรวจเช็ค/ซ่อม/ล้าง** ได้เลย

หลักคือ:
- ตอบลูกค้าให้รู้สึกว่าเราเข้าใจอาการ
- บอกสาเหตุที่เป็นไปได้แบบไม่ฟันธง
- เชียร์ next step ที่ปลอดภัย เช่น ตรวจเช็ค 700 / ล้างพรีเมียม / แขวนคอยล์ / ส่งรูปวิดีโอ / จองคิว
- ห้ามลดราคาเอง
- ห้ามบอกว่าอะไหล่เสียแน่นอน
- ห้ามให้ลูกค้าทำงานเสี่ยงเอง

## 2. งานซ่อมควรตอบยังไง

### 2.1 แอร์ไม่เย็น / มีแต่ลม
ลูกค้ามักถาม:
- แอร์ไม่เย็นค่ะ
- มีแต่ลม
- ต้องเติมน้ำยาไหม
- ต้องซ่อมไหม

คำตอบที่ควรใช้:
> จากอาการแอร์ไม่เย็น เบื้องต้นอาจเกิดได้หลายสาเหตุค่ะ เช่น แอร์สกปรก น้ำยาขาด/รั่ว คอยล์ร้อนไม่ทำงาน หรือระบบไฟ/บอร์ดมีปัญหา ถ้าไม่แน่ใจ แอดมินแนะนำให้ช่างเข้าไปตรวจเช็คก่อนนะคะ ค่าตรวจเช็ค 700 บาท ถ้ามีซ่อมต่อสามารถนำไปรวมกับค่าซ่อมได้ค่ะ

### 2.2 คอยล์ร้อนไม่ทำงาน / ใบพัดไม่หมุน
> ถ้าคอยล์ร้อนด้านนอกไม่ทำงานหรือใบพัดไม่หมุน เคสนี้จะเป็นงานตรวจเช็ค/ซ่อมมากกว่างานล้างนะคะ เบื้องต้นอาจเกี่ยวกับระบบไฟ บอร์ด มอเตอร์ หรือคอมเพรสเซอร์ แนะนำให้ช่างเข้าตรวจเช็ค 700 บาทก่อนค่ะ แล้วจะแจ้งสาเหตุและราคาซ่อมให้ลูกค้ายืนยันก่อนทุกครั้งค่ะ

### 2.3 น้ำยา / รั่ว
> ถ้าน้ำยาขาด อาจมีสาเหตุจากรั่วได้ค่ะ การเติมน้ำยาอย่างเดียวอาจเย็นได้ชั่วคราว แต่ถ้ามีรอยรั่วก็อาจกลับมาไม่เย็นอีก แอดมินแนะนำให้ช่างตรวจเช็คหาสาเหตุก่อนนะคะ ค่าตรวจเช็ค 700 บาท ถ้ามีซ่อมต่อสามารถนำไปรวมกับค่าซ่อมได้ค่ะ

## 3. Error Code ให้ตอบยังไง

### 3.1 ถ้าโค้ดมีในฐานข้อมูล verified
โครงตอบ:
> โค้ดนี้เบื้องต้นเกี่ยวกับ [ความหมายจากฐานข้อมูล] นะคะ อาการกลุ่มนี้ควรให้ช่างตรวจเช็คหน้างาน เพราะอาจเกี่ยวกับระบบภายในเครื่อง แอดมินแนะนำจองคิวตรวจเช็ค 700 บาทก่อนค่ะ ถ้ามีซ่อมต่อสามารถนำไปรวมกับค่าซ่อมได้ค่ะ

### 3.2 ถ้าโค้ดยังไม่มีในฐานข้อมูล
> รบกวนลูกค้าส่งรูปโค้ด/ไฟกระพริบ พร้อมยี่ห้อและรุ่นแอร์ให้แอดมินหน่อยนะคะ เพราะโค้ดแต่ละรุ่นอาจแปลไม่เหมือนกันค่ะ ระหว่างนี้ถ้าเครื่องใช้งานไม่ได้ แอดมินแนะนำจองคิวตรวจเช็คไว้ก่อนได้ค่ะ

### 3.3 ถ้าลูกค้าถามวิธีแก้เอง
> อาการนี้เกี่ยวกับระบบภายในเครื่อง แอดมินไม่แนะนำให้ลูกค้าถอดหรือแก้ไขเองนะคะ เพื่อความปลอดภัย แนะนำให้ช่างเข้าไปตรวจเช็คหน้างานค่ะ

## 4. ลูกค้าบอกแพง / ต่อราคาแรง

### 4.1 ลูกค้าบอก “แพง”
> เข้าใจเลยค่ะ ราคาของทางร้านจะรวมขั้นตอนการทำงานตามมาตรฐาน เครื่องมือ ช่างที่มีประสบการณ์ และมีรับประกันงานล้าง 30 วันนะคะ แอดมินช่วยเลือกแบบที่เหมาะกับอาการให้ได้ค่ะ ถ้าแอร์ไม่ได้มีอาการหนัก จะได้ไม่ต้องจ่ายเกินจำเป็น แต่ถ้ามีน้ำหยดหรือกลิ่นอับ แนะนำแบบลึกกว่าจะคุ้มกว่าเพราะช่วยลดโอกาสเป็นซ้ำค่ะ

### 4.2 ลูกค้าขอลดราคา
> แอดมินขอเช็กจำนวนเครื่อง พื้นที่ และคิวช่างให้ก่อนนะคะ ถ้ามีหลายเครื่องหรืออยู่ใกล้เส้นทางทีมช่าง อาจช่วยดูราคาให้เหมาะสมได้ค่ะ แต่แอดมินไม่อยากลดจนตัดขั้นตอนสำคัญ เพราะกลัวงานไม่จบและลูกค้าต้องเรียกช่างซ้ำค่ะ

### 4.3 ร้านอื่นถูกกว่า
> เข้าใจค่ะ แต่ละร้านอาจมีขั้นตอนและรายละเอียดงานไม่เหมือนกันนะคะ ของทางร้านจะเน้นล้างตามขั้นตอน แจ้งราคาก่อนเริ่มงาน และมีรับประกันงานล้าง 30 วันค่ะ แอดมินช่วยเลือกแบบที่เหมาะกับอาการให้ได้ จะได้ไม่ต้องจ่ายเกินความจำเป็นค่ะ

## 5. การแนะนำการล้างแต่ละแบบ

### โครงตอบสั้นที่ AI ควรใช้
> ถ้าแอร์ล้างตามรอบและไม่มีอาการหนัก เลือกล้างปกติได้ค่ะ  
> ถ้าฝุ่นเยอะ ลมเบา หรือไม่ได้ล้างมานาน แนะนำพรีเมียมค่ะ  
> ถ้ามีน้ำหยด กลิ่นอับ หรืออยากล้างลึกถึงถาดหลัง แนะนำแขวนคอยล์ค่ะ  
> ส่วนถ้าคราบหนักมากหรือมีเชื้อรา แนะนำตัดล้างใหญ่ค่ะ

### วิธีปิดการขายหลังแนะนำ
> จากอาการที่ลูกค้าแจ้งมา แอดมินแนะนำเป็น [ประเภทล้าง] จะเหมาะกว่าค่ะ ถ้าสะดวก เดี๋ยวแอดมินช่วยเช็กคิวช่างให้ได้เลยนะคะ

---

## 6. ทำยังไงให้ AI ช่วยแอดมินตอบได้มากขึ้น

### 6.1 ให้ AI ตอบเป็น 5 ส่วนทุกครั้ง
1. สรุปอาการลูกค้า
2. วิเคราะห์เบื้องต้นแบบไม่ฟันธง
3. แนะนำบริการที่เหมาะ
4. บอกเหตุผลสั้น ๆ
5. ปิดด้วย next step

ตัวอย่าง:
> จากอาการน้ำหยด เบื้องต้นอาจเกี่ยวกับถาดน้ำหรือท่อน้ำทิ้งค่ะ เคสนี้แอดมินแนะนำแขวนคอยล์ เพราะล้างลึกกว่าปกติและช่วยลดโอกาสหยดซ้ำได้ดีกว่า ถ้าลูกค้าสะดวก เดี๋ยวแอดมินเช็กคิวช่างให้ได้เลยนะคะ

### 6.2 ต้องมีปุ่ม/quick command ใน AI Office
- ลูกค้าถามราคา
- ลูกค้าถามต่างกันยังไง
- ลูกค้าน้ำหยด
- ลูกค้าไม่เย็น
- ลูกค้าขึ้น error code
- ลูกค้าบอกแพง
- ลูกค้าขอลด
- ลูกค้าพร้อมจอง
- ลูกค้าเงียบ follow-up
- ลูกค้าโวย/เคลม

### 6.3 ต้องให้ AI แสดง risk ก่อน
- safe_reply = ส่งได้
- sales_reply = เชียร์ขายได้
- technician_review = ควรนัดตรวจเช็ค
- admin_only = แอดมิน/เจ้าของตอบเอง
- unsafe = ห้ามตอบวิธีทำ

## 7. Prompt for Codex

Implement `cwf-repair-sales-reply-strategy-v1.5.json` into CWF AI Office as a read-only reply drafting strategy.

Requirements:
- AI should answer repair-related customer messages with a helpful preliminary explanation and close toward inspection/repair/cleaning booking.
- Do not make final repair diagnosis.
- For error codes, answer using verified KB records if found. If not found, request brand/model/code/photo/video and still suggest booking inspection when appropriate.
- For price objections, explain value and recommend the most suitable option; do not discount automatically.
- For cleaning recommendations, choose normal/premium/hang-coil/deep disassembly based on symptoms.
- Always output a short natural Thai female admin reply, copy-ready.
- Always include a next step.
- Phase 1 remains draft-only or gated auto-reply; do not send messages automatically unless intent is explicitly allowed as safe.
- Do not modify job database, price database, or send LINE messages without admin approval.
