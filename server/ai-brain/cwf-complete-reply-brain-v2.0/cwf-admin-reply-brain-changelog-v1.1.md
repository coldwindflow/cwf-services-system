# Change Log — CWF Admin Reply Brain v1.1

## Batch 011
เพิ่มตามข้อมูลผู้ใช้:
- เงื่อนไขราคาเพิ่มเติมสำหรับราคาเริ่มต้น/หน้างานยาก/เครื่องสกปรกมาก/ที่สูง/อุปกรณ์พิเศษ/ค่าจอดรถ/ทางด่วน/ค่าธรรมเนียมอาคาร
- กติกาว่าต้องแจ้งราคาและขอยืนยันลูกค้าก่อนซ่อมหรือเปลี่ยนอะไหล่ทุกครั้ง
- เงื่อนไขรับประกันและไม่รับประกัน
- ระบบ decision label สำหรับวิเคราะห์เคลม: likely_in_warranty, likely_out_of_warranty, needs_more_evidence, needs_technician_review
- ชุดวิเคราะห์อาการเสียแอร์เบื้องต้น เช่น ไม่เย็น น้ำหยด กลิ่นอับ เสียงดัง ไฟกระพริบ/error code
- โครงสร้างฐานข้อมูล error code แอร์ทุกยี่ห้อ
- วิธีให้ลูกค้าเช็คโค้ดเองแบบปลอดภัย
- ข้อห้ามไม่ให้ AI เดาโค้ดเองหรือให้ลูกค้าทำงานเสี่ยง
