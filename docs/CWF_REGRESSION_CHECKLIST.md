# CWF Regression Checklist

## งานแก้ไข / Revisit Schedule

- [ ] งานแก้ไขที่เลือกเวลาว่าง บันทึกเวลานัดได้
- [ ] งานแก้ไขที่เลือกเวลาชนกับงานอื่น ต้องถูก block
- [ ] ถ้าเวลาชน ต้องเห็น popup แจ้งให้ติดต่อแอดมิน/เลือกเวลาใหม่
- [ ] ถ้าเวลาชน ต้องไม่ขึ้นว่า “บันทึกนัดแล้ว”
- [ ] ถ้าเวลาชน ต้องไม่เปลี่ยน `appointment_datetime`
- [ ] หลังเลือกเวลาใหม่ที่ไม่ชน ต้องบันทึกได้
- [ ] งานปกติยังไม่ถูกกระทบ
- [ ] ปุ่มเก็บเงินลูกค้าไม่แสดงในงานแก้ไข
- [ ] งานแก้ไขยังไม่มีค่าตอบแทนเพิ่มเติมให้ช่าง

## Cache / PWA

ถ้าแก้ `app.js`:

- [ ] bump version ใน `tech.html`
- [ ] bump `CACHE_NAME` ใน `sw.js`
- [ ] update ASSETS ใน `sw.js`
- [ ] ทดสอบหลัง clear cache หรือเปิดใหม่
